# Claude Code 知見管理システム

このプロジェクトは Claude Code の知見管理システムを導入しています。

**最終更新**: 2025-12-26 (バッチ処理機能追加)

## 知見管理システム
このプロジェクトでは以下のファイルで知見を体系的に管理しています：

- `.claude/context.md`: プロジェクトの背景、目的、制約条件、技術スタック選定理由
- `.claude/project-knowledge.md`: 実装パターンや設計決定の知見
- `.claude/project-improvements.md`: 改善履歴・教訓
- `.claude/common-patterns.md`: 頻用コマンドパターン
- `.claude/debug-log.md`: 重要なデバッグ記録
- `.claude/debug/`: 一時的なデバッグファイル
